# DelegationAdapter
Java版DelegationAdapter
##使用说明
***Step 1. Add it in your root build.gradle at the end of repositories:
```bash
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
***Step 2. Add the dependency
```bash
	dependencies {
	        implementation 'com.github.liufx:delegationadapter:1.0.0'
	}
```
